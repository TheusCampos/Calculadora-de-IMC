# 🚀 Calculadora de IMC: Um Projeto de Estudo para Entender o Front-End Moderno!

Olá! Seja bem-vindo(a) a este projeto de estudo de uma **Calculadora de Índice de Massa Corporal (IMC)**. Meu objetivo aqui foi explorar e aplicar as melhores práticas de desenvolvimento Front-End, criando uma ferramenta simples, mas funcional e visualmente agradável.

Este projeto é ideal para quem quer ver como HTML, CSS e JavaScript podem trabalhar juntos para construir interfaces interativas e responsivas, com um toque de animação para melhorar a experiência do usuário.

## 🎯 O Que Você Encontra Aqui?

A Calculadora de IMC permite que você insira seu peso e altura para calcular seu Índice de Massa Corporal. Além do valor numérico, ela te dá uma classificação imediata (magreza, normal, sobrepeso ou obesidade) e um indicador visual na escala de IMC.

É uma forma rápida e intuitiva de ter uma ideia sobre sua saúde, sempre lembrando que o IMC é apenas um indicador e não substitui a avaliação de um profissional de saúde!

## ✨ Funcionalidades Destacadas (e o que aprendi com elas!)

-   **Cálculo Preciso de IMC:** A lógica por trás do cálculo é simples, mas a implementação me permitiu praticar a manipulação de inputs e a exibição de resultados dinamicamente com JavaScript.
-   **Classificação Instantânea e Visual:** A categorização do IMC e a atualização do "badge" de status e do indicador na escala foram ótimas para exercitar a lógica condicional e a manipulação de estilos CSS via JS.
-   **Design Intuitivo e Responsivo:** Desenvolvido com uma abordagem "mobile-first", este projeto me ajudou a aprofundar em CSS Grid e Flexbox para garantir que a interface se adapte perfeitamente a qualquer tamanho de tela.
-   **Animações Fluidas com GSAP:** Integrar a biblioteca GSAP foi um desafio divertido! Aprendi a criar transições suaves e animações que tornam a interação mais agradável, sem comprometer a performance.

## 🚀 Tecnologias Utilizadas (e por que as escolhi!)

Este projeto foi uma oportunidade para aplicar e consolidar conhecimentos em:

-   **HTML5 Semântico:** Foco em usar as tags HTML de forma correta para melhorar a acessibilidade e a estrutura do documento.
-   **CSS3 Moderno:**
    -   **BEM (Block, Element, Modifier):** Uma metodologia para organizar o CSS de forma escalável e fácil de entender.
    -   **Custom Properties (Variáveis CSS):** Para gerenciar cores e outros valores de estilo de forma centralizada e dinâmica.
    -   **CSS Grid e Flexbox:** As ferramentas essenciais para criar layouts complexos e responsivos.
-   **JavaScript (ES6+ Modular):** Organização do código em módulos para melhor manutenção e reusabilidade.
-   **GSAP (GreenSock Animation Platform):** Escolhida pela sua performance e flexibilidade para criar animações web de alta qualidade.

## 📁 Estrutura do Projeto (Minha Organização para um Código Limpo)

A organização do projeto segue uma estrutura lógica para facilitar o desenvolvimento e a manutenção:

-   `assets/css/`: Aqui você encontrará os arquivos CSS. Eu os organizei para incluir variáveis globais, um reset de estilos, estilos base e específicos para cada componente.
-   `assets/js/`: Contém os scripts JavaScript. O `script.js` é o arquivo principal que orquestra a lógica da calculadora e as interações.
-   `assets/img/`: Onde as imagens do projeto estão armazenadas.

## 🛠️ Como Rodar Este Projeto de Estudo Localmente

Quer ver como funciona ou explorar o código? É bem simples!

1.  **Clone o Repositório:**
    ```bash
    git clone https://github.com/seu-usuario/calculadora-imc.git
    cd calculadora-imc
    ```
    *(Lembre-se de substituir `seu-usuario` pelo seu nome de usuário no GitHub e `calculadora-imc` pelo nome real do repositório, se for diferente!)*

2.  **Abra com um Servidor Local:**
    Como este é um projeto puramente estático (HTML, CSS e JavaScript), você não precisa de configurações complexas de build. A maneira mais fácil é:
    -   Se você usa o **VS Code**, recomendo a extensão "Live Server". Basta clicar com o botão direito no arquivo `index.html` e selecionar "Open with Live Server".
    -   Você também pode usar qualquer outro servidor web local de sua preferência (como o `http-server` do Node.js, por exemplo).

## 🎨 Design, Acessibilidade e Performance

-   **Tema Dark Mode:** Um visual moderno e confortável, com cores de alto contraste que também ajudam na acessibilidade.
-   **Tipografia:** A fonte "Inter" do Google Fonts foi escolhida pela sua legibilidade e estética moderna.
-   **Foco em Acessibilidade:** Desde a semântica do HTML até o contraste de cores, busquei criar uma experiência inclusiva.
-   **Performance:** Otimizado para carregar rapidamente e funcionar de forma fluida, mesmo com as animações.
